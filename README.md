# hexl

Hexl is Hex to Plain Text converter and Plain Text to Hex converter made easy

# Installation

`git clone https://github.com/iinc0gnit0/hexl`

`cd hexl`

`chmod +x install.sh`

`./install.sh`

# Usage

use `./hexl` if you selected n during installation

use `hexl` if you selected y during installation

# Update

`git pull`
